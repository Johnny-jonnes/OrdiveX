PARTIE I - PRISE EN MAIN

Leçon 1 Installation et Premier Lancement

1.1 Qu'est-ce qu'OrdiveX ?

OrdiveX est un logiciel de gestion complet Conçu specialement pour les pharmacies. Il fonctionne directement dans votre navigateur web (Chrome, Edge) et peut être installe comme une application sur votre ordinateur ou telephone.

OrdiveX fonctionne meme sans connexion internet. Toutes vos données sont stockées localement sur votre appareil.

1.2 Installation sur ordinateur

Ouvrir Google Chrome ou Microsoft Edge

Taper l'adresse de votre pharmacie dans la barre d'adresse (fournie par TrillionX)

Une icone "Installer" apparait dans la barre d'adresse (ou une banniere en bas)

Cliquer sur "Installer" et confirmer

L'application apparait maintenant sur votre bureau comme un logiciel normal

Après l'installation, vous pouvez lancer OrdiveX en double-cliquant sur l'icone du bureau. Le navigateur n'apparait plus : c'est comme une application native.

1.3 Installation sur telephone ou tablette

Ouvrir Chrome (Android) ou Safari (iPhone/iPad)

Aller a l'adresse de votre pharmacie

Appuyer sur le menu (3 points sur Android, ou le bouton Partager sur iPhone)

Choisir "Ajouter a l'ecran d'accueil"

L'icone OrdiveX apparait sur votre ecran d'accueil

1.4 Configuration initiale (Onboarding)

Au premier lancement, OrdiveX vous guide etape par etape :

Nom de la pharmacie : Entrez le nom qui apparaitra sur les reçus et documents

Devise : Choisissez votre monnaie (GNF, XOF, MAD, EUR, USD)

Compte administrateur : Créez votre login et mot de passe

Cloud (optionnel) : Si vous avez plusieurs appareils, configurez la synchronisation

Leçon 2 Connexion, PIN et Verrouillage

2.1 Se connecter

Au lancement, l'ecran de connexion affiche la liste des utilisateurs

Cliquez sur votre nom

Entrez votre mot de passe

Cliquez "Connexion" ou appuyez sur Entrée

2.2 Code PIN

Chaque utilisateur doit definir un code PIN a 4 chiffres. Ce PIN sert a :

  • Deverrouiller rapidement l'ecran après une pause
  • Valider certaines operations sensibles (remise exceptionnelle, annulation)

Si vous n'avez pas encore de PIN, l'application vous demandera d'en créer un a votre prochaine connexion.

2.3 Verrouillage automatique

après 5 minutes d'inactivite (aucun mouvement de souris, clavier ou ecran tactile), l'ecran se verrouille automatiquement.

  • Entrez votre mot de passe pour reprendre votre travail
  • Votre panier n'est pas perdu : il est conserve en memoire
  • Vous pouvez aussi cliquer "Changer d'utilisateur" pour laisser un collegue se connecter

2.4 Se deconnecter

Cliquez sur le bouton deconnexion (icone de sortie) en bas de la barre laterale

Vous revenez a l'ecran de selection d'utilisateur

A gauche de l'ecran, la barre laterale affiche tous les modules accessibles selon votre role :

PrincipalTableau de Bord, Point de Vente, Caisse Journaliere
Stock et ProduitsCatalogue, Gestion des Stocks, Reapprovisionnement
Ordonnances et PatientsOrdonnances, Dossiers Patients

Sur telephone, la barre laterale est cachée. Appuyez sur le bouton menu (trois barres) en haut a gauche pour l'ouvrir.

3.2 La barre superieure

En haut de l'ecran, vous trouvez :

  • Bouton Menu : Ouvrir/fermer la barre laterale
  • Titre de la page : Indique ou vous êtes

En bas de la barre laterale, un point colore indique l'etat du reseau :

  • Point vert + "En ligne" : Connexion internet active, synchronisation fonctionnelle
  • Point rouge + "Hors ligne" : Pas d'internet, mais l'application continue de fonctionner normalement

Leçon 4 Le Tableau de Bord

4.1 Vue d'ensemble

Le tableau de bord est la premiere page qui apparait après la connexion. Il affiche un resume de l'activite de votre pharmacie :

  • Chiffre d'affaires du jour : Total des ventes realisées aujourd'hui
  • Nombre de ventes : Combien de transactions effectuées
  • Panier moyen : Montant moyen par vente
  • Produits en rupture : Nombre de produits avec un stock a zero

4.2 Graphiques

Le tableau de bord affiche des graphiques interactifs montrant :

  • L'evolution des ventes sur les 15 derniers jours
  • La repartition par mode de paiement (especes, mobile money, credit)
  • Les produits les plus vendus

4.3 Alertes rapides

Des cartes d'alerte s'affichent si :

  • Des produits sont en rupture de stock
  • Des lots arrivent a peremption dans les 90 jours
  • Des credits non rembourses sont en attente

Cliquez sur une alerte pour aller directement au module concerne.

PARTIE II - VENTES ET CAISSE

Leçon 5 Point de Vente (POS)

Le Point de Vente est le coeur d'OrdiveX. C'est ici que vous realisez toutes vos ventes quotidiennes.

5.1 Rechercher un produit

La barre de recherche en haut permet de trouver un produit par :

  • Nom commercial : ex. "Doliprane", "Amoxicilline"
  • Molecule (DCI) : ex. "paracetamol", "amoxicilline"
  • Code-barres : scannez ou tapez le code EAN
  • Code interne : le code que vous avez attribué au produit

La recherche filtre en temps réel pendant que vous tapez. Pas besoin d'appuyer sur Entrée.

Filtrer par categorie

Les pilules de categories en haut de la grille permettent de filtrer : Toutes, Antibiotiques, Antalgiques, Anti-inflammatoires, etc.

Trier les produits

Cliquez sur le bouton de tri pour changer l'ordre : Nom A-Z, Nom Z-A, Prix croissant, Prix decroissant, Stock croissant, Stock decroissant. Les produits en rupture sont toujours affiches en dernier.

5.2 Ajouter au panier

Trouvez le produit souhaité dans la grille

Cliquez sur la carte du produit (ou sur le bouton "Boite")

Le produit est ajouté au panier (panneau droit)

Si le produit est deja dans le panier, sa quantite augmente de 1

A chaque ajout, OrdiveX verifie automatiquement : le stock disponible, les interactions medicamenteuses avec les autres produits du panier, et les allergies du patient selectionné.

Comprendre les badges sur les cartes produit

BadgeSignification
Rx (bleu)Medicament sur ordonnance obligatoire
OTC (vert)Vente libre sans ordonnance
SC (rouge)Substance controlée - Traçabilite renforcée

5.3 gérer le panier

Pour chaque article dans le panier :

  • Boutons + / - : Augmenter ou diminuer la quantite
  • Bouton Poubelle : Retirer l'article du panier
  • Cliquer sur le prix : Modifier le prix pour une remise ponctuelle sur cet article

Remise globale

En bas du panier, le champ "Remise" permet de saisir un montant à deduire du total. Le TOTAL A PAYER est automatiquement recalculé.

5.4 Selectionner un patient

Dans le panneau droit, cliquez sur "Choisir un patient..."

Recherchez le patient par nom ou telephone

Ou cliquez "+ Nouveau" pour créer une fiche rapide

Selectionner un patient active :

  • La verification des allergies à chaque ajout au panier
  • La possibilite de vente à credit
  • La tracabilite complete sur le reçu

5.5 Lier une ordonnance

Cliquez sur "Lier ordonnance" dans le panneau droit

Sélectionnez une ordonnance existante ou créez-en une nouvelle

Les produits de l'ordonnance peuvent être ajoutes automatiquement au panier

5.6 Choisir le mode de paiement

ModeDescription
EspecesPaiement en liquide. Saisissez le montant reçu, la monnaie à rendre est calculée automatiquement
Mobile MoneyPaiement par Orange Money ou MTN MoMo. Entrez le numero de telephone
AssurancePrise en charge par un organisme d'assurance
MixteCombinaison de plusieurs modes (ex: partie especes + partie mobile money)
CreditLe patient paiera plus tard. Un patient DOIT être selectionné

5.7 Valider la vente

Verifiez que le panier est correct

Choisissez le mode de paiement

Cliquez sur "Valider" ou appuyez sur F5

OrdiveX effectue automatiquement :

  • Verification que le panier n'est pas vide
  • Verification que la caisse n'est pas cloturée
  • Verification du montant encaissé (pour les especes)
  • Creation de la vente dans l'historique
  • Destockage automatique FEFO (le lot qui expire en premier est utilise en premier)
  • Mise a jour de la caisse journaliere

5.8 Imprimer le reçu

après validation, un reçu s'affiche avec les boutons :

  • Imprimer A4 : Format standard pour imprimante de bureau
  • Imprimer Thermique : Format 80mm pour imprimante de caisse
  • Fermer : Fermer le reçu sans imprimer

5.9 Multi-onglets POS

Vous pouvez gérer plusieurs paniers en parallèle grâce aux onglets en haut du POS.

Cliquez sur "+" pour ouvrir un nouvel onglet

Chaque onglet a son propre panier, patient et ordonnance

Basculez entre les onglets en cliquant dessus

Utile quand un client part chercher de l'argent : ouvrez un nouvel onglet pour servir le client suivant, puis revenez au premier onglet.

5.10 Mettre un panier en attente

Preparez votre panier normalement

Cliquez sur le bouton "Attente" dans la barre d'actions

Le panier est sauvegardé avec le nom du préparateur

Pour retrouver les paniers en attente, cliquez sur le bouton "Paniers en attente"

Sélectionnez un panier pour le restaurer et continuer la vente

Les paniers en attente sont stockés localement sur cette machine. Ils ne sont pas visibles sur les autres machines de la pharmacie.

5.11 Déconditionnement (vente à l'unité)

Certains produits peuvent être vendus à l'unité (par plaquette ou par comprimé) :

Sur la carte du produit, cliquez sur "Plaquette" ou "Unité" au lieu de "Boite"

Le prix est automatiquement calculé au prorata

Le stock est déconditionné en conséquence

Le déconditionnement n'est disponible que pour les produits ou l'option a été activée dans le catalogue.

Leçon 6 Caisse Journaliere

6.1 Ouvrir la caisse

Allez dans Caisse Journaliere via la barre laterale

Si aucune caisse n'est ouverte, cliquez "Ouvrir la caisse"

Saisissez le fond de caisse (montant en especes au debut de la journée)

Confirmez l'ouverture

6.2 Entrées et sorties manuelles

En cours de journée, vous pouvez enregistrer :

  • Entrée manuelle : Un apport d'argent (ex: depot du gerant)
  • Sortie manuelle : Un retrait d'argent (ex: achat de fournitures, paiement fournisseur)

Cliquez sur "+ Entrée" ou "- Sortie"

Saisissez le montant et un motif

Confirmez

6.3 Cloture de caisse

En fin de journée, cliquez sur "Cloturer la caisse"

Saisissez le montant réel compte dans le tiroir-caisse

OrdiveX compare avec le montant theorique et affiche l'ecart

Confirmez la cloture

après la cloture, aucune vente ne peut être enregistrée pour cette journée. Si vous devez faire une vente urgente, il faut rouvrir une caisse.

Le rapport de cloture affiche :

  • Fond de caisse initial
  • Total des ventes par mode de paiement
  • Entrées et sorties manuelles
  • Montant theorique vs montant réel
  • Ecart (excedent ou deficit)

Leçon 7 Historique des Ventes

7.1 Consulter les ventes

Allez dans Historique Ventes via la barre laterale

La liste de toutes les ventes s'affiche, de la plus recente a la plus ancienne

Utilisez les filtres en haut pour chercher par date, vendeur ou mode de paiement

7.2 Détail d'une vente

Cliquez sur une vente pour voir son Détail

Vous verrez : la liste des produits, les quantites, le patient, l'ordonnance liée, le vendeur, le preparateur (si different), le mode de paiement et le montant.

7.3 Reimprimer un reçu

Dans le Détail d'une vente, cliquez sur "Reimprimer"

Choisissez le format (A4 ou thermique)

7.4 Credits en cours

Les ventes a credit apparaissent avec un badge "Credit". Pour enregistrer un remboursement :

Cliquez sur la vente a credit

Cliquez sur "Enregistrer remboursement"

Saisissez le montant rembourse

Leçon 8 Retours de Medicaments

8.1 Enregistrer un retour

Allez dans Retours Medicaments via la barre laterale

Cliquez sur "+ Nouveau retour"

Sélectionnez le produit retourne

Indiquez la quantite et le motif du retour

Choisissez l'action : Remise en stock (si le produit est en bon etat) ou Destruction (si le produit est endommage ou perime)

Confirmez le retour

8.2 Suivi des retours

La liste des retours affiche tous les retours enregistres avec :

  • Date et heure du retour
  • Produit concerne
  • Quantite retournée
  • Motif
  • Action effectuée (remise en stock ou destruction)
  • Utilisateur ayant traite le retour

Leçon 10 Gestion des Stocks

10.1 Entrée de stock (reception de marchandise)

Allez dans Gestion des Stocks

Cliquez sur "+ Entrée de stock"

Sélectionnez le produit

Saisissez la quantite reçue

Renseignez le numero de lot et la date de peremption

Optionnel : indiquez le fournisseur et le prix d'achat

Confirmez l'entrée

10.2 Lots et dates de peremption (FEFO)

OrdiveX gere les stocks par lot. Chaque lot a :

  • Un numero de lot (inscrit sur l'emballage)
  • Une date de peremption
  • Une quantite

FEFO (First Expired, First Out) : lors d'une vente, OrdiveX utilise automatiquement le lot qui expire en premier. Vous n'avez rien a faire, c'est automatique.

10.3 Inventaire et ajustements

Si le stock réel ne correspond pas au stock dans OrdiveX :

Trouvez le produit dans la Gestion des Stocks

Cliquez sur "Ajuster"

Saisissez la quantite réelle

Indiquez le motif (casse, vol, erreur de comptage, etc.)

Confirmez l'ajustement

Leçon 11 Reapprovisionnement

Allez dans Reapprovisionnement via la barre laterale ou le bouton rapide en haut

La liste des produits dont le stock est en dessous du seuil d'alerte s'affiche

Pour chaque produit, vous voyez : stock actuel, seuil, quantite suggerée

Cochez les produits a commander

Cliquez sur "Generer bon de commande" pour créer automatiquement un bon de commande fournisseur

PARTIE IV - PATIENTS ET ORDONNANCES

Leçon 12 Dossiers Patients

12.1 Créer un dossier patient

Allez dans Dossiers Patients

Cliquez sur "+ Nouveau patient"

Remplissez : Nom, Prenom, Telephone, Date de naissance, Allergies connues

Enregistrez

12.2 Consulter l'historique d'un patient

Cliquez sur le nom du patient dans la liste

Vous verrez : ses informations personnelles, ses allergies, l'historique de toutes ses ventes, ses ordonnances et ses credits en cours.

Les allergies saisies dans le dossier patient sont verifiées automatiquement a chaque vente au POS. Si un produit contient une molecule a laquelle le patient est allergique, une alerte rouge apparait.

Allez dans Ordonnances

Leçon 14 Interactions Medicamenteuses et Allergies

OrdiveX verifie automatiquement 30+ interactions medicamenteuses a chaque ajout au panier :

  • Alerte rouge : Interaction dangereuse — la vente est bloquée sauf validation par un pharmacien
  • Alerte orange : Precaution d'emploi — un avertissement s'affiche
  • Alerte allergie : Le patient selectionne est allergique a un composant du produit

Ces verifications se font sans internet. La base d'interactions est stockée localement.

StatutSignification

EnvoyeCommande transmise au fournisseur
reçuMarchandise reçue et stockée

16.3 Exporter des commandes

Dans Bons de Commande, cliquez sur "Exporter" en haut à droite

Un fichier JSON contenant toutes vos commandes est téléchargé

Vous pouvez aussi exporter une seule commande : ouvrez-la avec "Voir", puis cliquez "Exporter cette commande"

Le fichier exporté peut être réimporté sur une autre machine OrdiveX ou conservé comme archive.

16.4 Importer des commandes

Dans Bons de Commande, cliquez sur "Importer"

Sélectionnez un fichier .json (format OrdiveX) ou .csv

OrdiveX analyse le fichier et vous demande confirmation

Les commandes sont créées automatiquement

Si une commande importée a le statut "Reçue", le stock est automatiquement mis à jour : les lots sont créés et les quantités ajoutées au stock.

Gestion automatique des erreurs à l'import

SituationComportement d'OrdiveX

Produit non trouvé dans le catalogueCommande créée mais stock non mis à jour pour cet article
Fichier mal forméMessage d'erreur clair affiché
Fichier vide ou trop volumineuxRejeté avec message

Si vous préférez préparer vos commandes dans Excel :

ColonneDescriptionObligatoire

supplierName / fournisseurNom du fournisseurNon

Enregistrez votre Excel en CSV avec séparateur point-virgule (;). OrdiveX détecte automatiquement le séparateur.

16.6 Imprimer un bon de commande

Ouvrez le bon de commande et cliquez sur "Imprimer"

PARTIE VI - ANALYTIQUE

Leçon 17 Métriques Business

Ce module fournit des indicateurs avances pour piloter votre pharmacie :

  • Chiffre d'affaires : par jour, semaine, mois avec graphiques
  • Marges : marge brute par produit et globale
  • Top ventes : produits les plus vendus
  • Produits dormants : produits avec peu ou pas de ventes
  • Repartition paiements : especes vs mobile money vs credit
  • Performance vendeurs : CA par vendeur

Leçon 18 Traçabilite et Pharmacovigilance

Ce module permet de tracer chaque medicament depuis sa reception jusqu'a sa delivrance :

  • Recherche par lot : retrouvez tous les patients ayant reçu un lot specifique
  • Recherche par patient : retrouvez tous les lots delivres a un patient
  • Rappels de lots : identifiez rapidement les patients concernes par un rappel
  • Substances controlées : registre special pour les stupefiants et psychotropes

PARTIE VII - OUTILS

Leçon 19 Centre d'Impression

Le Centre d'Impression centralise tous les documents imprimables :

  • reçus de vente (A4 ou thermique 80mm)
  • Bons de commande
  • Rapport de cloture de caisse
  • Etiquettes de prix
  • Fiches patient
  • Inventaire

Allez dans Centre d'Impression, Sélectionnez le type de document, ajustez les paramêtres, et cliquez "Imprimer"

Leçon 29 Questions Frequentes (FAQ)

Je n'arrive plus a me connecter, j'ai oublie mon mot de passe

Contactez votre administrateur ou le support TrillionX. Le mot de passe peut être reinitialise sans perdre vos données.

L'application affiche "Hors ligne" alors que j'ai internet

Fermez et rouvrez l'application. Si le probleme persiste, verifiez votre connexion WiFi et rechargez la page avec Ctrl+Shift+R.

J'ai fait une vente par erreur, comment l'annuler ?

Allez dans Historique des Ventes, trouvez la vente et enregistrez un retour dans Retours de Medicaments. Le stock sera remis automatiquement.

Le stock affiche n'est pas le bon

Allez dans Gestion des Stocks, trouvez le produit et cliquez "Ajuster" pour corriger le stock. Indiquez le motif de l'ecart.

Comment imprimer sur mon imprimante thermique ?

Allez dans Paramêtres et Sélectionnez "Thermique 80mm" comme format par defaut. Assurez-vous que votre imprimante est connectée et configurée dans Windows.

Les données sont-elles en securite ?

Oui. Toutes les données sont stockées localement sur votre appareil dans une base de données chiffrée (IndexedDB). Si la synchronisation cloud est activée, les données sont egalement sauvegardées sur des serveurs securises.

Combien d'appareils puis-je utiliser ?

Autant que vous voulez. Avec la synchronisation cloud activée, tous les appareils partagent les memes données en temps réel.
